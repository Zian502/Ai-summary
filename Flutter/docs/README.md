# Flutter 学习文档

本目录包含 Flutter 从入门到进阶的完整学习文档，所有 API 均基于 Flutter 最新版本。

## 文档结构

1. [快速入门](./01-快速入门.md) - Flutter 简介、环境搭建、第一个应用
2. [Dart 语言基础](./02-Dart语言基础.md) - Dart 语法、数据类型、函数、类等
3. [Widget 基础](./03-Widget基础.md) - Widget 概念、常用组件、组件树
4. [布局和样式](./04-布局和样式.md) - 布局组件、样式装饰、响应式设计
5. [状态管理](./05-状态管理.md) - StatefulWidget、状态提升、Provider、Riverpod
6. [路由和导航](./06-路由和导航.md) - Navigator、命名路由、路由守卫
7. [网络请求](./07-网络请求.md) - http 包、dio、JSON 解析、错误处理
8. [数据持久化](./08-数据持久化.md) - SharedPreferences、SQLite、文件存储
9. [动画和交互](./09-动画和交互.md) - 动画控制器、过渡动画、手势识别
10. [平台交互](./10-平台交互.md) - 平台通道、原生插件、设备功能调用
11. [性能优化](./11-性能优化.md) - 渲染优化、内存管理、构建优化
12. [测试](./12-测试.md) - 单元测试、Widget 测试、集成测试
13. [打包和发布](./13-打包和发布.md) - Android/iOS 打包、应用签名、发布流程

## 文档特点

- ✅ **API 对齐最新版本** - 所有 API 均基于 Flutter 最新稳定版
- ✅ **JavaScript 类比** - 每个概念都有对应的 JavaScript/React 类比，便于理解
- ✅ **丰富的示例** - 每个知识点都配有完整的 Demo 代码和详细注释
- ✅ **应用场景** - 结合实际应用场景，学以致用
- ✅ **易于理解** - 循序渐进，从基础到进阶

## 学习建议

1. **按顺序学习** - 建议按照文档顺序学习，前面的知识是后续章节的基础
2. **动手实践** - 每个示例代码都建议自己动手敲一遍，加深理解
3. **查阅官方文档** - 本文档作为学习指南，遇到问题时查阅 [Flutter 官方文档](https://flutter.cn/docs)
4. **项目实战** - 学完基础后，建议通过实际项目巩固知识

## 前置知识

- 基本的编程概念（变量、函数、类等）
- 了解 JavaScript/TypeScript 或类似语言会更容易理解
- 了解移动应用开发概念（可选）
