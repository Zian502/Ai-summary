# 模式匹配

模式匹配是 Rust 最强大的特性之一，它允许你比较值与一系列模式，然后根据匹配的模式执行相应的代码。Rust 的模式匹配是穷尽的（exhaustive），意味着必须处理所有可能的情况。

## match 表达式

`match` 是 Rust 中最强大的控制流结构，它允许你将一个值与一系列模式进行比较，并根据匹配的模式执行代码。

### 基本语法

```rust
enum Coin {
    Penny,
    Nickel,
    Dime,
    Quarter,
}

fn value_in_cents(coin: Coin) -> u8 {
    match coin {
        Coin::Penny => 1,
        Coin::Nickel => 5,
        Coin::Dime => 10,
        Coin::Quarter => 25,
    }
}

fn main() {
    let coin = Coin::Quarter;
    println!("硬币价值: {} 分", value_in_cents(coin));
}
```

### 绑定值的模式

```rust
#[derive(Debug)]
enum UsState {
    Alabama,
    Alaska,
    // ... 其他州
}

enum Coin {
    Penny,
    Nickel,
    Dime,
    Quarter(UsState),  // 关联数据
}

fn value_in_cents(coin: Coin) -> u8 {
    match coin {
        Coin::Penny => 1,
        Coin::Nickel => 5,
        Coin::Dime => 10,
        Coin::Quarter(state) => {
            println!("州季度币来自 {:?}!", state);
            25
        }
    }
}

fn main() {
    let coin = Coin::Quarter(UsState::Alaska);
    value_in_cents(coin);
}
```

### 匹配 Option<T>

```rust
fn plus_one(x: Option<i32>) -> Option<i32> {
    match x {
        None => None,
        Some(i) => Some(i + 1),
    }
}

fn main() {
    let five = Some(5);
    let six = plus_one(five);
    let none = plus_one(None);
    
    println!("{:?}, {:?}", six, none);
}
```

### 匹配是穷尽的

```rust
fn plus_one(x: Option<i32>) -> Option<i32> {
    match x {
        Some(i) => Some(i + 1),
        // 如果注释掉这一行，编译器会报错
        // None => None,  // 错误！必须处理所有情况
    }
}
```

### 通配模式和 _ 占位符

```rust
fn main() {
    let dice_roll = 9;
    match dice_roll {
        3 => add_fancy_hat(),
        7 => remove_fancy_hat(),
        other => move_player(other),  // 绑定到变量
    }
    
    // 使用 _ 占位符（不绑定值）
    match dice_roll {
        3 => add_fancy_hat(),
        7 => remove_fancy_hat(),
        _ => reroll(),  // 忽略值
    }
    
    // 使用 _ 忽略所有值
    match dice_roll {
        3 => add_fancy_hat(),
        7 => remove_fancy_hat(),
        _ => (),  // 什么都不做
    }
}

fn add_fancy_hat() {}
fn remove_fancy_hat() {}
fn move_player(num_spaces: u8) {}
fn reroll() {}
```

## if let 语法糖

`if let` 是 `match` 的语法糖，用于只关心一种模式而忽略其他模式的情况。

### 基本用法

```rust
fn main() {
    let some_value = Some(3u8);
    
    // 使用 match
    match some_value {
        Some(3) => println!("三"),
        _ => (),
    }
    
    // 使用 if let（更简洁）
    if let Some(3) = some_value {
        println!("三");
    }
    
    // if let else
    let mut count = 0;
    if let Coin::Quarter(state) = coin {
        println!("州季度币来自 {:?}!", state);
    } else {
        count += 1;
    }
}
```

### 与其他语言对比

| 语言 | 模式匹配 | 穷尽性检查 | 语法糖 |
|------|---------|-----------|--------|
| **Rust** | `match` | ✅ 编译时 | `if let` |
| **Haskell** | `case` | ✅ 编译时 | `if` |
| **Scala** | `match` | ✅ 编译时 | `if` |
| **Swift** | `switch` | ✅ 编译时 | `if case` |
| **C++** | `switch` | ❌ | 无 |
| **Java** | `switch` | ❌ | 无 |

**Rust 特点**：
- 编译时保证穷尽性
- 模式可以绑定值
- `if let` 提供简洁语法

## while let 条件循环

`while let` 允许在循环中使用模式匹配。

```rust
fn main() {
    let mut stack = Vec::new();
    
    stack.push(1);
    stack.push(2);
    stack.push(3);
    
    // 使用 while let 弹出栈
    while let Some(top) = stack.pop() {
        println!("{}", top);
    }
}
```

## for 循环中的模式

```rust
fn main() {
    let v = vec!['a', 'b', 'c'];
    
    // enumerate 返回 (index, value)
    for (index, value) in v.iter().enumerate() {
        println!("{}: {}", index, value);
    }
    
    // 解构元组
    let points = [(0, 0), (1, 2), (3, 4)];
    for (x, y) in points.iter() {
        println!("点: ({}, {})", x, y);
    }
}
```

## let 语句中的模式

```rust
fn main() {
    // 解构元组
    let (x, y, z) = (1, 2, 3);
    println!("x: {}, y: {}, z: {}", x, y, z);
    
    // 解构结构体
    struct Point {
        x: i32,
        y: i32,
    }
    
    let p = Point { x: 0, y: 7 };
    let Point { x, y } = p;
    println!("x: {}, y: {}", x, y);
    
    // 部分解构
    let Point { x, .. } = p;
    println!("x: {}", x);
}
```

## 函数参数中的模式

fn main() {

## 可反驳和不可反驳模式

### 不可反驳模式（Irrefutable Patterns）

总是匹配成功的模式，用于 `let`、`for`、函数参数等。

```rust
let x = 5;  // x 总是匹配成功
```

### 可反驳模式（Refutable Patterns）

可能匹配失败的模式，用于 `if let`、`while let`、`match` 等。

```rust
if let Some(x) = some_option {  // 可能匹配失败
    // ...
}
```

## 模式语法

### 匹配字面量

```rust
fn main() {
    let x = 1;
    
    match x {
        1 => println!("一"),
        2 => println!("二"),
        3 => println!("三"),
        _ => println!("其他"),
    }
}
```

### 匹配命名变量

```rust
fn main() {
    let x = Some(5);
    let y = 10;
    
    match x {
        Some(50) => println!("得到 50"),
        Some(y) => println!("匹配，y = {:?}", y),  // y 是新的变量
        _ => println!("默认情况，x = {:?}", x),
    }
    
    println!("最后：x = {:?}, y = {}", x, y);
}
```

### 多个模式

```rust
fn main() {
    let x = 1;
    
    match x {
        1 | 2 => println!("一或二"),
        3 => println!("三"),
        _ => println!("其他"),
    }
}
```

### 匹配范围

```rust
fn main() {
    let x = 5;
    
    match x {
        1..=5 => println!("一到五"),
        6..=10 => println!("六到十"),
        _ => println!("其他"),
    }
    
    // 字符范围
    let c = 'c';
    match c {
        'a'..='j' => println!("前半部分字母"),
        'k'..='z' => println!("后半部分字母"),
        _ => println!("其他字符"),
    }
}
```

### 解构结构体

```rust
struct Point {
    x: i32,
    y: i32,
}

fn main() {
    let p = Point { x: 0, y: 7 };
    
    match p {
        Point { x, y: 0 } => println!("在 x 轴上，x = {}", x),
        Point { x: 0, y } => println!("在 y 轴上，y = {}", y),
        Point { x, y } => println!("不在轴上，({}, {})", x, y),
    }
}
```

### 解构枚举

```rust
enum Message {
    Quit,
    Move { x: i32, y: i32 },
    Write(String),
    ChangeColor(i32, i32, i32),
}

fn main() {
    let msg = Message::ChangeColor(0, 160, 255);
    
    match msg {
        Message::Quit => {
            println!("Quit 变体没有数据")
        }
        Message::Move { x, y } => {
            println!("移动到 x: {}, y: {}", x, y);
        }
        Message::Write(text) => println!("文本消息: {}", text),
        Message::ChangeColor(r, g, b) => {
            println!("改变颜色为 RGB({}, {}, {})", r, g, b);
        }
    }
}
```

### 解构嵌套的结构体和枚举

```rust
enum Color {
    Rgb(i32, i32, i32),
    Hsv(i32, i32, i32),
}

enum Message {
    Quit,
    Move { x: i32, y: i32 },
    Write(String),
    ChangeColor(Color),
}

fn main() {
    let msg = Message::ChangeColor(Color::Hsv(0, 160, 255));
    
    match msg {
        Message::ChangeColor(Color::Rgb(r, g, b)) => {
            println!("改变颜色为 RGB({}, {}, {})", r, g, b);
        }
        Message::ChangeColor(Color::Hsv(h, s, v)) => {
            println!("改变颜色为 HSV({}, {}, {})", h, s, v);
        }
        _ => (),
    }
}
```

### 解构结构体和元组

```rust
fn main() {
    let ((feet, inches), Point { x, y }) = ((3, 10), Point { x: 3, y: -10 });
    println!("{} 英尺 {} 英寸，点 ({}, {})", feet, inches, x, y);

```

### 忽略模式中的值

```rust
fn foo(_: i32, y: i32) {
    println!("只使用 y 参数: {}", y);
}

fn main() {
    foo(3, 4);
    
    // 忽略元组的某些部分
    let numbers = (2, 4, 8, 16, 32);
    match numbers {
        (first, _, third, _, fifth) => {
            println!("一些数字: {}, {}, {}", first, third, fifth);
        }
    }
    
    // 使用 .. 忽略剩余部分
    struct Point {
        x: i32,
        y: i32,
        z: i32,
    }
    
    let origin = Point { x: 0, y: 0, z: 0 };
    match origin {
        Point { x, .. } => println!("x 是 {}", x),
    }
}
```

### 匹配守卫（Match Guards）

额外的 `if` 条件，用于更复杂的匹配。

```rust
fn main() {
    let num = Some(4);
    
    match num {
        Some(x) if x < 5 => println!("小于 5: {}", x),
        Some(x) => println!("{}", x),
        None => (),
    }
    
    // 使用外部变量
    let x = 4;
    let y = false;
    
    match x {
        4 | 5 | 6 if y => println!("yes"),
        _ => println!("no"),
    }
}
```

### @ 绑定

`@` 允许在创建一个变量的同时测试它是否匹配某个模式。

```rust
enum Message {
    Hello { id: i32 },
}

fn main() {
    let msg = Message::Hello { id: 5 };
    
    match msg {
        Message::Hello { id: id_variable @ 3..=7 } => {
            println!("找到范围内的 id: {}", id_variable);
        }
        Message::Hello { id: 10..=12 } => {
            println!("找到另一个范围内的 id");
        }
        Message::Hello { id } => {
            println!("找到其他 id: {}", id);
        }
    }
}
```

### 示例 3：错误处理

```rust
use std::fs::File;
use std::io::ErrorKind;

fn main() {
    let f = File::open("hello.txt");
    
    let f = match f {
        Ok(file) => file,
        Err(error) => match error.kind() {
            ErrorKind::NotFound => match File::create("hello.txt") {
                Ok(fc) => fc,
                Err(e) => panic!("创建文件时出错: {:?}", e),
            },
            other_error => {
                panic!("打开文件时出错: {:?}", other_error);
            }
        },
    };
}
```

## 总结

- **match 表达式**：强大的模式匹配，必须处理所有情况
- **if let**：用于只关心一种模式的简洁语法
- **while let**：在循环中使用模式匹配
- **模式语法**：字面量、变量、范围、解构、守卫等
- **穷尽性**：编译器保证所有情况都被处理
- **可反驳性**：区分总是匹配和可能匹配失败的模式
