## 🚀 Python 简介

Python 是一种高级、解释型、面向对象的编程语言，由 Guido van Rossum 在 1991 年首次发布。

### Python 特点

- **简洁优雅**：语法简洁，代码可读性强
- **易于学习**：学习曲线平缓，适合初学者
- **功能强大**：丰富的标准库和第三方库
- **跨平台**：支持 Windows、Linux、macOS 等平台
- **动态类型**：无需声明变量类型，提高开发效率
- **解释执行**：无需编译，直接运行
- **面向对象**：支持面向对象编程范式
- **社区活跃**：庞大的开发者社区和丰富的资源

### 与其他语言的对比

| 特性 | Python | Java | C++ | JavaScript |
|------|--------|------|-----|------------|
| 类型系统 | 动态类型 | 静态类型 | 静态类型 | 动态类型 |
| 执行方式 | 解释执行 | 编译到字节码（JVM） | 编译型 | 解释执行（JIT） |
| 内存管理 | GC | GC | 手动/智能指针 | GC |
| 语法简洁度 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐ | ⭐⭐⭐⭐ |
| 学习难度 | ⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |
| 性能 | ⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| 应用领域 | Web、数据科学、AI | 企业应用、Android | 系统编程、游戏 | Web 前端、后端 |

### Python 适用场景

- **Web 开发**：Django、Flask 等框架
- **数据科学**：NumPy、Pandas、Matplotlib
- **人工智能**：TensorFlow、PyTorch、Scikit-learn
- **自动化脚本**：系统管理、任务自动化
- **网络爬虫**：Scrapy、BeautifulSoup
- **游戏开发**：Pygame
- **桌面应用**：Tkinter、PyQt
- **科学计算**：SciPy、SymPy

### Python 不适用场景

- **系统级编程**：操作系统内核、设备驱动（建议使用 C/C++）
- **实时性要求极高的应用**：高频交易系统（解释执行有延迟）
- **内存受限环境**：嵌入式系统（Python 运行时占用内存较大）

## 📝 快速开始

```bash
# 检查 Python 版本
python --version
# 或
python3 --version
# 输出：Python 3.12.x

# 检查 pip 版本
pip --version
# 或
pip3 --version
# 输出：pip 24.x.x

# 创建第一个 Python 程序
mkdir my-python-app
cd my-python-app

# 创建 hello.py
echo "print('Hello, Python!')" > hello.py

# 运行程序
python hello.py
# 或
python3 hello.py
```
